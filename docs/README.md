**这是什么？**


这是一个xposed模块。


主要功能包含一键指令和自动指令，类似于ColorOS的小布指令，或MIUI的自动任务，或者Tasker等自动任务应用。

相对其他同类软件，ShortX提供了更多系统级触发器与操作。
